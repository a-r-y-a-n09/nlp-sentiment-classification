# Sentiment Analysis using VADER, TF-IDF and ML (In Progress)

This project focuses on building a sentiment prediction system by combining lexicon-based, classical machine learning, and planned transformer-based approaches.

## 🚧 Status

This project is currently under development. Improvements and additional features are being actively worked on.

---

## 📌 Overview

The goal of this project is to classify sentiment (positive/negative) using:

* Lexicon-based analysis with VADER
* Text vectorization using TF-IDF
* Machine learning models for prediction
* Planned integration of transformer models such as RoBERTa

---

## ⚙️ Features

* Data preprocessing and cleaning
* Handling missing values
* Text feature extraction using TF-IDF
* Sentiment scoring using VADER
* Model training using Scikit-learn
* Evaluation using standard metrics
* Planned experimentation with RoBERTa for contextual sentiment understanding

---

## 🧠 Tech Stack

* Python
* Pandas, NumPy
* Scikit-learn
* NLTK (VADER Sentiment Analyzer)
* Planned: RoBERTa (Transformer-based NLP model)
